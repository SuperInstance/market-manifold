## 3. The "World-Class" UX Standard
All repos must move from "Technical Specs" to "Engineer-First Experiences."

**The Standard:**
- **README.md**: 
  - 🚀 **The Hook**: What is this? Why does it matter? (Single sentence)
  - 🛠️ **Quickstart**: $3 \to 5$ commands to get it running.
  - 📐 **The la-link**: A visual ASCII/Mermaid diagram of the flow.
  - 📚 **The Knowledge Path**: a $\to$ b $\to$ c onboarding guide.
- **TUTORIALS.md**: "I want to do X" $\to$ a step-by-step "How to" with minimal code.
- **TEMPLATES/**: Boilerplate for new plugins/agents/blocks.
- **SAMPLES/**: "The Golden Path" — a perfect implementation example.
